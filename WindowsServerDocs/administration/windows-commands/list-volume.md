# <a name="list-volume"></a>list volume

すべてのディスク上のベーシック ボリュームとダイナミック ボリュームの一覧を表示します。

## <a name="syntax"></a>構文

```
list volume
```

### <a name="examples"></a>例

検出されたボリュームを一覧表示するには、次のように入力します。

```
list volume
```

*ボリューム 2*を削除するには、次のように入力します。

```
delete volume=2
```
