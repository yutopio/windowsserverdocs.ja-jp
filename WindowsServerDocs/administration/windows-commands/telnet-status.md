# <a name="telnet-status"></a>telnet: 状態

> 適用対象: Windows Server (半期チャネル)、Windows Server 2019、Windows Server 2016、Windows Server 2012 R2、Windows Server 2012

ステータス情報を表示します。

## <a name="syntax"></a>構文

```
st[atus]
```

## <a name="example"></a>例

ステータス情報を表示するには、次のように入力します。

```
st
```
